# voice-interview-coach

AI-powered interview simulator with voice input, real-time scoring, and role-specific question sets.

## 🛠 Tech Stack
- Java 17 / Spring Boot 3
- React / TailwindCSS
- Web Speech API
- OpenAI API
- MariaDB
